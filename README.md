# Xiaomi Korea Community

Photo community web app built with Vite, React, TypeScript, and Tailwind CSS.

## Getting started

```sh
npm install
npm run dev
```

## Scripts

- `dev`: start local dev server
- `build`: production build
- `preview`: preview production build
- `lint`: run ESLint
- `test`: run Vitest

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui

## Deployment

Build the app and serve the `dist` folder with your preferred static host.
