# 샤오미코리아 커뮤니티 웹사이트 UI/UX 상세 명세서

## 1. 프로젝트 개요

### 1.1 목적

- 샤오미코리아 회원들을 위한 사진 중심 커뮤니티 웹사이트 구축
- 레퍼런스: Leica Club (http://www.leicaclub.net/) 구조 참고
- 기본 디자인 위주, 커스터마이징 최소화

### 1.2 핵심 가치

- **간결성**: 불필요한 장식 제거, 콘텐츠 중심
- **접근성**: 모든 사용자가 쉽게 이용 가능
- **확장성**: 향후 기능 추가 용이한 구조

---

## 2. 솔루션 제안

### 2.1 권장 플랫폼: **워드프레스 (WordPress)**

#### 선정 이유

1. **비용 효율성**: 오픈소스, 무료 테마/플러그인 다수
2. **커뮤니티 기능 강화**: BuddyPress, bbPress 등 플러그인 활용
3. **갤러리 최적화**: NextGEN Gallery, Envira Gallery 등 전문 플러그인
4. **관리 편의성**: 직관적인 관리자 대시보드
5. **유지보수**: 국내 개발자 풀 넓음, 하자보수 용이

#### 대안 플랫폼 비교

| 플랫폼          | 장점                                | 단점                                  | 적합도                |
| --------------- | ----------------------------------- | ------------------------------------- | --------------------- |
| **WordPress**   | 커뮤니티/갤러리 플러그인 풍부, 저렴 | 보안 관리 필요                        | ⭐⭐⭐⭐⭐            |
| 아임웹          | 드래그앤드롭 편리, 한국형           | 커스터마이징 제한적, 갤러리 기능 약함 | ⭐⭐⭐                |
| 고도몰          | 이커머스 특화                       | 커뮤니티 기능 부족                    | ⭐⭐                  |
| 카페24          | 호스팅+쇼핑몰                       | 커뮤니티 목적엔 과함                  | ⭐⭐                  |
| **커스텀 개발** | 완전 맞춤형                         | 비용/시간 多                          | ⭐⭐⭐ (예산 충분 시) |

#### 추천 구성

- **CMS**: WordPress 6.x
- **테마**: Astra / GeneratePress (가볍고 커스터마이징 용이)
- **필수 플러그인**:
  - **갤러리**: NextGEN Gallery / Envira Gallery
  - **커뮤니티**: bbPress (포럼) + BuddyPress (회원 프로필)
  - **SEO**: Yoast SEO
  - **보안**: Wordfence Security
  - **성능**: WP Rocket / W3 Total Cache
  - **댓글**: wpDiscuz (고급 댓글 기능)
